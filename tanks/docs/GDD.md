# tanky — GDD v1.0 (2026-07-03 확정)

> diep.io류 싱글플레이 탱크 아레나 스코어어택. 팀원들이 URL로 접속해 한 판씩 즐기고 랭킹을 겨룬다.
> 원작과 구분: 이름·아트 전부 자체 제작 (메커니즘만 참조). 멀티플레이는 비목표 — AI 봇이 대체한다.

## 확정 결정 (2026-07-03 사용자 승인)
1. **싱글플레이 + AI 봇** — 실시간 멀티 없음. 봇 탱크가 아레나에서 파밍/교전.
2. **클래스는 티어 2까지** — Lv15에서 트윈/스나이퍼/머신건/플랭크 중 택1. 스탯 8종.
3. **별도 프로젝트** — boom과 폴더·서버·배포 전부 분리 (`tanks/`).

## 코어 루프
움직인다(WASD) → 조준·사격(마우스) → 도형을 깨서 XP → 레벨업·스탯 찍기 → Lv15 클래스 선택 →
봇 탱크와 교전 → 죽으면 점수 제출 → "한 판 더".
목숨 1개 (diep 문법). 점수 = 획득 XP 총합.

## 화면·월드
- 캔버스 960×600 고정, 월드 3200×3200. 카메라는 플레이어 추종(월드 경계 클램프).
- 격자 배경 + 월드 경계선. 우하단 미니맵(플레이어·봇·오각형 위치).

## 성장
- XP 곡선: `need(L) = xpBase × L^xpPow` (기본 6 × L^1.5), 최대 Lv30. Lv15 누적 ≈ 1,900 XP.
- 레벨당 스탯 포인트 1. 스탯 8종 × 최대 5포인트: 회복 / 최대체력 / 몸통대미지 / 탄속 / 관통(탄HP) / 탄대미지 / 연사 / 이동속도.
- 숫자키 1~8 또는 HUD 클릭으로 분배.

## AI 봇
- 상태머신: **파밍**(가까운 도형) → **교전**(시야 520px 내 플레이어, 조준 오차 ±0.12rad) → **도주**(HP 25% 미만).
- 시간에 따라 스폰 레벨 상승 `min(2 + t/35, 22)`, 스탯 자동 분배, Lv15+는 랜덤 클래스.
- 동시 최대 4기, 스폰은 플레이어 700px 밖. 죽으면 12초 후 보충.

## "내 잘못" 원칙 (boom 계승)
- 봇 스폰은 시야 밖 + 스폰 직후 2초 무적(반투명 표시).
- 봇 최대 4기 — 다구리 상한. 조준 오차로 초탄 회피 가능.
- 모든 대미지원은 화면에서 식별 가능 (탄환은 발사자 색).

## 난이도 상한
봇 레벨 캡 22, 동시 봇 4, 도형 총량 70 고정. 무한 스케일 없음.

## 리더보드 4종
score(XP총합) / level(도달 레벨) / kills(봇 킬) / survival(생존초). 이름당 베스트 1개.

## 밸런스 의도
- 첫 레벨업까지 ~10초(사각형 1개), Lv15(클래스)까지 ~2분 30초, 평균 런 4~6분.
- 오각형은 "가면 이득, 봇도 노린다" — 리스크/리워드 허브.
- 몸통박치기 빌드(체력+몸통대미지 몰빵)도 유효하게 — 빌드 다양성이 리플레이 동력.

## 비목표
실시간 멀티, 티어3+ 클래스, 드론/트랩류, 모바일 최적화(터치 이동만 지원), 안티치트.
