# boom (amnesia) — 코어 기획 v2.2
> v2.1 + 2026-07-03 확정 결정 반영: 하이브리드 룰 / 듀얼 입력 / Node 단일 서버.
> 설계 원칙: **모든 오브젝트는 서로 다른 "의사결정"을 강제해야 존재 이유가 있다.**
> 색 문법: **푸른 계열 = 이득 / 붉은 계열 = 위협.** 색만 보고 0.2초 내 판단.

## 0. 확정 룰 (v2.1과 다른 점)
- **하이브리드**: 기본 90초 타임어택 + 라이프 3.
  - 시간 종료 → **클리어**: 잔여 라이프 × 50점 보너스.
  - 라이프 소진 → 즉시 종료 (사망 원인 표시).
  - **타임 부스터 ⏱ (신규, 푸른 계열 6번째)**: 희귀 스폰, +15초. 총 시간 최대 180초 캡. → "더 벌 것인가, 더 오래 버틸 위험을 질 것인가"라는 새 결정 축.
- **입력 듀얼**: 마우스/터치 추종(lerp 0.22, 700px/s 클램프) + 키보드 화살표/WASD(430px/s). 최근 입력 우선.
- 피격 시: 라이프 -1, 1.5초 무적 + 깜빡임, 콤보 리셋. (즉사 아님 → v2.1 "사망 순간" 연출은 라이프 소진 시 적용)

## 3. 등장 스케줄 (난이도 = 시간)
| 시간 | 신규 등장 |
|---|---|
| 0s | 오브, 드론 |
| 15s | 코발트, 헌터 |
| 30s | 스플리터, 마그넷 |
| 50s | 불릿, 프리즈 |
| 70s | 마인, 스타, **타임 부스터** |

→ 70초(풀 로스터) 도달이 단기 목표. 타임 부스터가 70s에 열려 "클리어 직전 연장 유혹" 형성.

## 4. 템포 파라미터 (config.defaults.json이 원본 — 여기는 의도만)
- 위협 기본 속도 150 / 속도 계수 `min(1+t/45, ×2.2)` / 스폰 계수 `min(1+t/30, ×3.0)` / 스폰 간격 하한 0.28s / 동시 위협 40 / 오브 공급 0.7s / 플레이어 클램프 700px/s.
- 무한 스케일 금지 — 후반 난이도는 속도가 아니라 **밀도**로.

## 5. 점수 & 콤보
- 콤보: 3초 내 연속 수집 시 +1 (×1~×5). 3초 공백 또는 피격 시 리셋. 콤보 게이지는 플레이어 근처 표시.
- 니어미스: 위협 반경 30px 스침 +2, **위협 1개당 1회만** (마인 파밍 방지).
- 점수 = (수집 기본점 × 콤보 × 스타 배수) + 니어미스 + 생존 1초당 +1 + 클리어 시 잔여 라이프 × 50.
- **러시 모드 (차별화 후크)**: 수집 25개마다 4초간 무적 + 자석(160px). 발동 직전 "한 개 더" 무리수 유도.

## 6. "한 판 더" 루프
1. 종료 순간 0.3초 슬로모 + 원인 표시 ("HUNTER에게 잡힘" / "CLEAR!")
2. 결과: 점수 / 베스트 / **"베스트까지 -N점"** / 최대 콤보 / 생존시간 / 니어미스
3. NEW BEST 시 연출 강화 (색 반전)
4. 클릭 1번 0.5초 내 재시작. 이름 재입력 없음(localStorage).

## 7. 리더보드 (팀 공유, 서버 저장) — 4탭
최고 점수(메인) / 최장 생존 / 최대 콤보 / 니어미스. 이름 기준, 이름당 베스트만 노출.

## 8. 첫 화면 & 자세히보기 유도
- 첫 화면: 이름 입력 + 모드 선택(현재 CLASSIC 1종, 구조는 확장 가능) + 3줄 요약 설명 + START.
- **자세히보기 유도 장치**: ① 첫 방문 시 자동 오픈 ② 미열람 유저에겐 버튼에 펄스 배지 ③ 모달 안에 "오브젝트 도감" — 색·효과를 미리 알면 점수가 오르는 정보로 구성해 실익 부여.

## 9. 관리자 콘솔 (/admin.html)
- config 전체 항목 자동 렌더링 → 숫자 수정 → 저장 → 다음 판부터 반영 (배포 없이 라이브 밸런싱).
- 랭킹 리셋 버튼. `x-admin-key` 헤더 보호.
