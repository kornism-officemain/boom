> - 플레이어 = 아기 발광어 (초롱 루어, 꼬리 스윙, 좌우 플립, 콤보 5+ 시 입 벌림). 색 문법 유지: 푸른 먹이 / 붉은 포식자.
> - 매핑: 오브→발광 플랑크톤, 코발트→은빛 치어, 프리즈/마그넷→냉기·소용돌이 진주, 부스터→황금진주★·무지개진주✦, 드론→독가시 성게, 헌터→아기상어, 스플리터→분열 해파리, 레이저→작살(경고선=낚싯줄), 마인→기뢰복어(페이드인=부풀기 전 무해).
> - 배경: 심연 그라디언트 + 마린 스노우(침강) + 빛기둥 + 상승 기포. 러시 = "광란의 포식". 봇 시드도 심해 네이밍.

## 0. 확정 룰 (v3.0)
- **퓨어 서바이벌**: 라이프 3, 시간 무제한. 라이프 소진 = 종료. 점수 상한 없음 → 리더보드 경쟁이 실력 상한까지 무한히 열림.
  - **하트 ♥ (타임부스터 대체)**: 희귀 스폰, 라이프 +1 (최대 4). 풀피일 땐 +20점. → "한 번만 더 버티면 하트가 나올지도"가 런을 붙잡는 손잡이.
  - **마일스톤 보너스**: 30초 생존마다 +30/+60/+90... (선형 증가). 중간 목표 제공.
  - **후반 압박**: 90초부터 헌터 동시 3기. 속도 상한(×2.2)은 유지 — 후반 난이도는 속도가 아닌 밀도·구성으로 (공정성).
- **입력 듀얼**: 마우스/터치 추종(lerp 0.22, 700px/s 클램프) + 키보드 화살표/WASD(430px/s). 최근 입력 우선.
- 피격 시: 라이프 -1, 1.5초 무적 + 깜빡임, 콤보 리셋.
- **긴박 연출**: 마지막 라이프 1일 때 화면 붉은 비네트 펄스 + 라이프 HUD 점멸. (타이머가 없어진 대신 긴장의 원천 = 목숨)
- **HUD**: 점수(오를 때 크기별 확대 연출 3단계) / 중앙 위험도 게이지(🕐 생존초 표시, 난이도 상승 따라 파랑→초록→빨강, 상한 도달 시 펄스) / 라이프.

## 3. 등장 스케줄 (v3: 초반 루즈함 해소 — 압축)
| 시간 | 신규 등장 |
|---|---|
| 0s | 오브, 드론(소행성) |
| 10s | 코발트, 헌터 |
| 20s | 스플리터, 마그넷 |
| 35s | 불릿(레이저), 프리즈 |

→ 50초 풀 로스터 도달이 1차 목표, 90초(헌터 3기)부터가 "진짜 게임". 해금마다 화면 상단 **NEW 배너**로 진행감 제공.

## 4. 템포 파라미터 (config.defaults.json이 원본 — 여기는 의도만)
- 위협 기본 속도 150 / 속도 계수 `min(1+t/45, ×2.2)` / 스폰 계수 `min(1+t/30, ×3.0)` / 스폰 간격 하한 0.28s / 동시 위협 40 / 오브 공급 0.7s / 플레이어 클램프 700px/s.
- 무한 스케일 금지 — 후반 난이도는 속도가 아니라 **밀도**로.

## 5. 점수 & 콤보
- 콤보: 3초 내 연속 수집 시 +1 (×1~×5). 3초 공백 또는 피격 시 리셋. 콤보 게이지는 플레이어 근처 표시.
- 니어미스: 위협 반경 30px 스침 +2, **위협 1개당 1회만** (마인 파밍 방지).
- 점수 = (수집 기본점 × 콤보 × 스타 배수) + 니어미스 + 생존 1초당 +1 + 마일스톤(30초마다 +30씩 증가).
- **러시 모드 (차별화 후크)**: 수집 25개마다 4초간 무적 + 자석(160px). 발동 직전 "한 개 더" 무리수 유도.

## 6. "한 판 더" 루프
1. 종료 순간 0.3초 슬로모 + 원인 표시 ("HUNTER에게 잡힘" / "CLEAR!")
2. 결과: 점수 / 베스트 / **"베스트까지 -N점"** / 최대 콤보 / 생존시간 / 니어미스
3. NEW BEST 시 연출 강화 (색 반전)
4. 클릭 1번 0.5초 내 재시작. 이름 재입력 없음(localStorage).

## 7. 리더보드 (팀 공유, 서버 저장) — 4탭
최고 점수(메인) / 최장 생존 / 최대 콤보 / 니어미스. 이름 기준, 이름당 베스트만 노출.

## 8. 첫 화면 & 자세히보기 유도
- 첫 화면: 이름 입력 + 모드 선택(현재 CLASSIC 1종, 구조는 확장 가능) + 3줄 요약 설명 + START.
- **자세히보기 유도 장치**: ① 첫 방문 시 자동 오픈 ② 미열람 유저에겐 버튼에 펄스 배지 ③ 모달 안에 "오브젝트 도감" — 색·효과를 미리 알면 점수가 오르는 정보로 구성해 실익 부여.

## 9. 관리자 콘솔 (/admin.html)
- config 전체 항목 자동 렌더링 → 숫자 수정 → 저장 → 다음 판부터 반영 (배포 없이 라이브 밸런싱).
- 랭킹 리셋 버튼. `x-admin-key` 헤더 보호.
